# TechComm AI 🌍 - Engineers' English Training Platform

**TechComm AI** は、海外のチームと協業する日本のエンジニアのために開発された、**実践的な技術英語学習アプリケーション**です。

一般的な英会話アプリとは異なり、「バグ報告」「仕様の確認」「コードレビュー」など、ソフトウェア開発の現場で実際に発生するシチュエーションに特化したトレーニングを提供します。

---

## 📖 どんなアプリ？（概要）

「英語のドキュメントを読むのに時間がかかる」「チャットで技術的な質問をしたいけれど、どう書けば失礼にならないかわからない」
そんなエンジニアの悩みを、最新のAI（Google Gemini）が解決します。

このアプリは**あなたの専属のネイティブ技術英語コーチ**です。24時間いつでも、あなたのレベルに合わせた課題を出題し、書いた英文をその場で添削してくれます。

---

## ✨ 主な機能

### 1. 📧 Reading Comprehension（読解トレーニング）
**「現場のメールやチャット」を読み解く力を養います。**
*   **リアルな素材:** 教科書的な文章ではなく、海外エンジニアから送られてくるような少し崩れた表現や専門用語（API, Latency, PRなど）を含んだメールが生成されます。
*   **無限の問題数:** AIがその都度新しい文章を作成するため、二度と同じ問題は出ません。
*   **丁寧な解説:** 正解・不正解の理由を日本語で分かりやすく解説します。

### 2. ✍️ Technical Writing（記述トレーニング）
**「伝わる技術文書」を書く力を養います。**
*   **シチュエーション提示:** 「本番DBのCPU使用率が急増していることをチームに報告せよ」といった具体的な指令が出されます。
*   **AIによる即時添削:** あなたが書いた英文に対し、AIが100点満点で採点します。
    *   **ネイティブ版の提示:** 「ネイティブのエンジニアならこう書く」という理想的な例文を表示。
    *   **日本語での講評:** 文法ミスや、より適切な言い回しを日本語でアドバイスします。

### 3. 🏆 レベルアップシステム
**ゲーム感覚で成長を実感できます。**
*   **Lv 1〜50の段階設定:** 初級（基本的な報告）から始まり、中級（技術的な議論）、上級（マネジメント・交渉）へとステップアップします。
*   学習履歴が保存され、日々の成長が可視化されます。

## 🚀 このアプリのすごいところ

| 特徴 | 一般的な英語学習 | TechComm AI |
| :--- | :--- | :--- |
| **学習内容** | 旅行、日常会話、ビジネス一般 | **開発現場、バグ報告、仕様策定** |
| **教材** | 固定されたテキスト | **AIが無限に生成（飽きない）** |
| **フィードバック** | 正解/不正解のみ | **「なぜダメか」「どう直すべきか」を解説** |
| **対象** | 全員 | **ソフトウェアエンジニア特化** |

## 🛠 技術スタック（開発者向け情報）

このアプリケーションは最新のWeb技術と生成AIを組み合わせて構築されています。

*   **Frontend:** React (TypeScript), Vite
*   **Styling:** Tailwind CSS (PostCSS)
*   **AI Model:** Google Gemini 3 Flash (via Google GenAI SDK)
*   **Icons:** Lucide React
*   **State Management:** React Hooks + LocalStorage

## 💻 始め方（Installation）

このプロジェクトをローカル環境で実行する手順です。

1. **リポジトリのクローン**
   ```bash
   git clone [repository-url]
   cd techcomm-ai
   ```

2. **依存関係のインストール**
   ```bash
   npm install
   ```

3. **環境変数の設定**
   `.env.example` をコピーして `.env` ファイルを作成し、APIキーを設定してください。
   ```bash
   cp .env.example .env
   ```
   ```env
   API_KEY=AIzaSyDa-xxxxxxxxxxxxxxxxxxxxxxxx
   ```
   **注意:** `.env` ファイルはGitにコミットしないでください（`.gitignore`で除外設定済みです）。

4. **アプリケーションの起動**
   ```bash
   npm run dev
   ```
   ブラウザで `http://localhost:5173` にアクセスしてください。
